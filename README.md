###################
Sistem Informasi Laboratorium QC
###################

Sistem informasi sekaligus manajemen data di laboratorium divisi Quality Control Pabrik Gula Kebon Agung.
Berbasis PHP Codeigniter 3 dan Bootstrap 5.
DBMS menggunakan MariaDB.
Sebagian data masuk database melalui media Human Machine Interface (HMI), sehingga tidak sepenuhnya dioperasikan oleh manusia. 
Sistem ini ditujukan untuk melakukan monitoring, koreksi data karena kesalahan HMI, pelaporan data analisa per jam, per hari atau per shift, serta melakukan pencetakan Certificate of Analysis (COA).
